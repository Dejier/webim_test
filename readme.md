ID ВК приложения 6217922
Тип - standalone
Для запуска - auth.html. Больше ничего не требуется.

Авторизует пользователя с правами на доступ к друзьям и показывает 5 случайных друзей(имя и фамилия).
Токен и данные авторизованного пользователя записываются в local storage, очищаются при выходе из программы.
При последующем запуске проверяет наличие данных в хранилище. Запрос 5 рандомных друзей происходит всегда, данные по ним не сохраняются.
При первом входе для авторизации нужно скопировать access_token из адресной строки в открывшемся окне. 
Пример url:
https://oauth.vk.com/blank.html#access_token=f2b364363464364780b1bc49b661191fe8734587a80cc01a451b475e1610b5420143bb1d1fbcd&expires_in=86400&user_id=10008864
И сам токен f2b364363464364780b1bc49b661191fe8734587a80cc01a451b475e1610b5420143bb1d1fbcd(для примера), который нужно вставить в текстовое поле.
