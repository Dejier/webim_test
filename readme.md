ID ВК приложения 6217922
Тип - standalone
Для запуска - auth.html. Больше ничего не требуется.

Авторизует пользователя с правами на доступ к друзьям и показывает 5 случайных друзей(имя и фамилия).
Токен и данные авторизованного пользователя записываются в local storage, очищаются при выходе из программы.
При последующем запуске проверяет наличие данных в хранилище. Запрос 5 рандомных друзей происходит всегда, данные по ним не сохраняются.
